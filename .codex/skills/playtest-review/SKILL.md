# Playtest Review Skill

Output:

- Keep
- Change
- Confusing parts
- Too easy / too hard
- Bugs noticed
- Next tiny build ticket
- Parking lot ideas

Kid-friendly question set:

- What did you like?
- What felt weird?
- What should happen next?

Keep the review short. Turn the next idea into one tiny build ticket.
