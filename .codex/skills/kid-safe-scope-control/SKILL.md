# Kid-Safe Scope Control Skill

Use this when an idea is too large, vague, risky, overwhelming, or likely to derail the session.

Do:

- Preserve the idea.
- Explain the tiny first version.
- Move large parts to `docs/idea-parking-lot.md`.
- Avoid saying the idea is bad.
- Produce one small playable step.
- Protect momentum.

Examples:

Big idea:
Make a huge world.

Tiny first version:
Add one place name.

Big idea:
Add a big challenge.

Tiny first version:
Add one simple obstacle.
