# Docs Update Skill

Update relevant docs after a meaningful change.

Rules:

- Update docs with concise notes.
- Preserve child wording when possible.
- Record what was actually built, not what was merely discussed.
- Add oversized ideas to the parking lot.
- Keep the Game Bible current.

Docs to consider:

- `docs/game-bible.md`
- `docs/session-log.md`
- `docs/prompt-history.md`
- `docs/playtest-notes.md`
- `docs/idea-parking-lot.md`
