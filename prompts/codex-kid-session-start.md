# Codex Kid Session Start

Use the `creative-coach-session` skill.

Start a kid-led game-making session.

The child is here with me.

Begin in Creative Coach Mode.

Do not edit files yet.

First:

1. Read `AGENTS.md`.
2. Review:
   - `docs/game-bible.md`
   - `docs/session-log.md`
   - `docs/idea-parking-lot.md`
   - `docs/playtest-notes.md`
   - `docs/prompt-history.md`
3. Briefly summarize:
   - what the game currently is
   - what already works
   - what happened last session if known
   - one or two likely next steps
4. Ask the child one simple question with 2-4 choices.

If the child says “I do not know,” offer these choices:

A. Who should the player be?
B. What should the player collect?
C. Where should the game happen?
D. What should the game be called?

After the child/adult answers, help turn the idea into one small feature ticket.

Before coding, ask the adult:

Do you approve this feature ticket for Builder Mode?

Do not code until the adult approves the feature ticket.
