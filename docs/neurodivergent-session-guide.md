# Neurodivergent-Friendly Session Guide

This guide supports ADHD/autistic-friendly game-making sessions. It is not medical advice. The goal is to make the creative process predictable, comfortable, and child-led.

## Good Session Shape

Use the same pattern often:

1. First, recap the game.
2. Then, ask one choice question.
3. Then, make one tiny feature ticket.
4. Then, the adult approves or pauses.
5. Then, Codex builds.
6. Then, the child playtests.
7. Then, save next ideas.

## Helpful Defaults

- One question at a time.
- 2-4 choices.
- Short sentences.
- Concrete options.
- Visible progress.
- Clear stop points.
- Breaks are allowed.
- Stopping early can be success.

## Response Options

The child can respond by:

- Saying an answer.
- Typing an answer.
- Pointing to a choice.
- Picking A, B, C, or D.
- Saying “skip.”
- Saying “pause.”
- Asking the adult to translate.

## Avoid

- Long lectures.
- Too many choices.
- Surprise rule changes.
- Pressure to decide fast.
- Forced brainstorming from a blank page.
- Treating silence as refusal.
- Treating movement as not listening.
- Flashing visuals or sudden sounds.

## If Attention Drops

Try one of these:

- Offer two choices instead of four.
- Ask for a point or letter answer.
- Switch to playtest for one minute.
- Save the idea and stop.
- Ask: “Should we pause or make one tiny change?”

## If the Idea Gets Huge

Keep the idea. Write it in `docs/idea-parking-lot.md`.

Then ask:

Which tiny first piece should we build?

A. A look
B. A sound
C. A rule
D. A name

## If the Child Gets Frustrated

Use fewer words.

Try:

First, we save the idea.

Then, we stop or pick one tiny piece.

Do not argue about the idea. Protect the relationship and the next session.

## Sensory Comfort

Default to:

- No flashing.
- No jump scares.
- No sudden audio.
- No forced timers.
- Calm colors unless the child asks otherwise.
- Adjustable difficulty when possible.

## Adult Checkpoint

Before Builder Mode, the adult should confirm:

- The feature is small.
- The child still wants it.
- The change is comfortable.
- The session can end after this build.
