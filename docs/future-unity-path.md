# Future Unity Path

This starter begins web-first so families can make visible progress quickly with simple files: HTML, CSS, and vanilla JavaScript.

Unity may make sense later when the child has practiced the AI Director loop and wants richer animation, physics, 3D worlds, controller support, or app-store style publishing.

A future Unity repo could add:

- Scenes and prefabs.
- 2D or 3D physics.
- Sprite animation.
- Sound and particle systems.
- Larger project organization.

Suggested progression:

1. Build small web games.
2. Learn the AI Director loop.
3. Pick a favorite idea.
4. Rebuild or expand it in Unity later.

Unity is not part of this repo.
